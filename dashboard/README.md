# Streamlit Dashboard for Medical Data Analyser

This directory contains the web-based dashboard for the Medical Data Analyser.

## Running the Dashboard

```bash
streamlit run app.py
```

The dashboard will be available at `http://localhost:8501`

## Features

- **Patient Data Upload**: Upload CSV files with patient health data
- **Data Overview**: View summary statistics and data quality metrics
- **Trend Analysis**: Interactive trend visualization for selected patients
- **Patient Comparison**: Compare health metrics across multiple patients
- **Predictions**: View disease risk predictions
- **Exports**: Download analysis results and visualizations
